# Portfolio Website

Website portfolio pribadi yang menampilkan profil, proyek, dan cara menghubungi saya.

## Deskripsi
Portfolio ini dibuat untuk menampilkan:
- Informasi pribadi
- Skill dan keahlian
- Proyek-proyek yang pernah dikerjakan
- Form kontak

## Teknologi yang Digunakan
- HTML5
- CSS3

## Cara Instalasi
1. Clone repository ini:
```bash
   git clone https://github.com/rholans/TA-Praktikum-WEB.git
```
2. Buka file `index.html` di browser

## Cara Penggunaan
Buka file `index.html` di browser favorit Anda.
